# Noosh.

A single-page website for Noosh., a small-batch home bakery owned by Anooshey.

## Project overview

Pure HTML/CSS/JS — no framework, no build step. Everything lives in `index.html`.

## Brand & aesthetic

- **Palette**: warm cream (`#f8e8d2`), peach (`#f4c8b6`), cherry red (`#c4615a`), sage green (`#a4b894`), plus deep brown/cocoa tones for text
- **Style**: hand-drawn, cute, cozy — inspired by Studio Ghibli food art
- **Fonts**: Fraunces (display/headings), Caveat (handwritten accents), Nunito (body) — all via Google Fonts
- **Illustrations**: all inline SVG, no external image assets
- **Brand voice**: warm, friendly, lowercase — feels like a note from a friend, not a brand

## Design rules

- Preserve the overall vibe. Don't make significant visual or layout changes without checking first.
- Keep the handcrafted, small-batch feeling in every element.
- New UI elements should match the existing palette and hand-drawn style.
- Inline SVG only for illustrations — no raster images or icon libraries.
- CSS custom properties are defined in `:root` — always use them, don't hardcode colors or fonts.
